Method

Each Stacks reward cycle is 2,100 Bitcoin blocks — a 2,000-block reward phase followed by a 100-block prepare phase. Cycle 0 begins at block 666,050.

Past dates are confirmed Bitcoin block timestamps from mempool.space. Forecasts use the live block interval of the current difficulty epoch.
